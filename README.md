# Actifyme - Sports & Wellness Event Management

A modern, dark-theme, single-page static website for an event management brand that organizes sports tournaments and yoga/wellness events.

## Features

- **Dark Theme UI** with charcoal/black backgrounds
- **Dual Accent Colors**: Red/orange for sports, green/teal for yoga
- **Sticky Header** with smooth scroll navigation
- **Responsive Design** - Mobile-first approach
- **Smooth Animations** and hover effects
- **Modern Typography** using Poppins and Inter fonts

## Sections

1. **Hero Section** - Full-screen with split visual design
2. **About Section** - Brand introduction with mission/vision
3. **Services Section** - Card-based layout with hover effects
4. **Events Section** - Upcoming and past events
5. **Contact Section** - Contact form with WhatsApp/Call buttons
6. **Footer** - Links and social media icons

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

4. Preview production build:
```bash
npm run preview
```

## Tech Stack

- **React 18** - UI library
- **Vite** - Build tool and dev server
- **CSS3** - Styling with custom properties and animations

## Customization

### Colors

Edit CSS variables in `src/index.css`:
- `--accent-sports`: Sports theme color (default: #ff4444)
- `--accent-yoga`: Yoga theme color (default: #00d4aa)
- `--bg-primary`: Main background (default: #0a0a0a)
- `--bg-secondary`: Secondary background (default: #1a1a1a)

### Contact Information

Update contact details in `src/components/Contact.jsx`:
- WhatsApp number in `handleWhatsApp` function
- Phone number in `handleCall` function
